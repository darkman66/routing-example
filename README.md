# routing-example
install via pipenv

```bash
$ pip install pipenv
```
# run it

1. server

Start it first

```bash
$ python server.py
```

2. router

```sh
$ python router.py
```

3. Client


 * client 1

```sh
$ python client.py --client-id 1
```

* client 2

```sh
$ python client.py --client-id 2
```

* client 3

```sh
$ python client.py --client-id 3
```
